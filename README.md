# Enhanced canvas theme for Obsidian

## TL; NR

The color template of `Enhanced Canvas` theme in vscode + the interaction interface of `Obsidian Nord` theme.

## Motivation

This is a theme transplanted from the `Enhanced Canvas` theme on vscode([Enhanced Canvas Theme - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=NicolasMendes.enhanced-canvas-theme)). Since I love that color-style for its vitality, proper abstraction and clearness with unique tone, I decided to do this transplant with the help of Codex.

The interface interaction is inhereted from `Obsidian Nord` theme to retain the sense of cleanness and modernness.
